## Overview
**Forza Horizon 6 Spotify Radio** is the ultimate Spotify integration mod for Forza Horizon 6. Listen to your favorite Spotify playlists, personalized radio, and music in real-time while racing in the open world. Perfect Spotify in-game music experience with seamless integration.

> [!Important]
> This tool requires a Spotify Premium account and a legitimate copy of Forza Horizon 6.

## Installation
### For Regular Users
1. Download and extract the archive
2. Run the installer or executable as Administrator
3. Log in with your Spotify account
4. Launch Forza Horizon 6
5. Enjoy your personal Spotify music while racing

## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| Spotify not connecting     | Check internet connection and Spotify login   |
| No sound in game           | Ensure Spotify is playing and not paused      |
| App doesn't start          | Run as Administrator                          |
| Antivirus warning          | Add the folder to exclusions                  |
| Music stops                | Restart both Spotify and the game             |
